# Curso-Agentes-I.A.-IEBS
Repositorio Curso Agentes I.A.
## Propósito del agente

Este proyecto implementa un agente inteligente simple que interactúa con el usuario desde la consola.  
El agente puede responder preguntas básicas y simular información del clima para distintas ciudades.

## Herramienta utilizada

Se utilizó **GitHub Copilot** dentro de **Visual Studio Code** para generar el código del agente en Python.

## Flujo lógico del agente

1. El usuario escribe una pregunta en la consola.
2. El agente analiza el texto ingresado.
3. Si detecta una ciudad, genera un clima simulado.
4. Si detecta otra intención (hora o nombre), responde en consecuencia.
5. El agente devuelve la respuesta al usuario.
